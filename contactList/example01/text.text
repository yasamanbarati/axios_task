what is axios?
برای ارسال و دریافت درخواست های  HTTP درخواست های HTTP ممکن  است از بک اند یک اپلیکیشن یا یک third party api باشند.
 از axios می توان برای رهگیری درخواست ها (request ها) و پاسخ های (response های) HTTP استفاده کرد
  و امنیت client-side را در برابر XSRF نیز فعال می کند. هم چنین قابلیت لغو درخواست ها را نیز دارد.
